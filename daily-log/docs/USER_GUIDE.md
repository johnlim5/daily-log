# Daily Log ユーザーガイド

毎日の習慣を記録・分析するためのシンプルなアプリです。

## はじめに

Daily Logは、日々の習慣（ルーチン）を記録し、継続状況を可視化するアプリです。
朝・昼・夜の時間帯別に習慣を管理し、AIによる分析でより良い習慣形成をサポートします。

---

## クイックスタート

### 1. ログイン
アプリにアクセスすると、パスワード入力画面が表示されます。
- 管理者から共有されたパスワードを入力してください
- 「パスワードを保存する」にチェックを入れると、次回以降自動ログインできます

### 2. 習慣を記録する
1. 画面下部の「記録」タブを選択
2. 朝・昼・夜のタブで時間帯を切り替え
3. 完了した習慣をタップすると、チェックマークと完了時刻が記録されます
4. 時刻をタップすると、記録時刻を修正できます

### 3. 習慣を管理する
1. 「管理」タブを選択
2. 既存の習慣名をタップして編集
3. 絵文字をタップして変更（または🤖ボタンでAI提案）
4. 「+ 追加」ボタンで新しい習慣を追加
5. 上下ボタンで並び順を変更
6. 「×」ボタンで習慣を削除

## 画面説明

### 記録タブ
日々の習慣を記録するメイン画面です。2つの表示モードがあります。

#### 一覧モード（デフォルト）
| 要素 | 説明 |
|------|------|
| 朝/昼/夜タブ | 時間帯を切り替えます（現在時刻に応じて自動選択） |
| 進捗バー | 完了数/全体数とプログレスバーを表示 |
| 一覧/集中ボタン | 表示モードを切り替え |
| 習慣リスト | タップで完了/未完了を切り替え |
| NEXTマーク | 次にやるべきタスクをハイライト表示 |
| 完了時刻 | タップすると時刻を修正できます |

#### 集中モード（フォーカスモード）
1つのタスクに集中できるカード形式の表示です。

| 要素 | 説明 |
|------|------|
| タスクカード | 次のタスクを大きく表示 |
| 完了ボタン | タスクを完了として記録 |
| 後でボタン | このタスクを後回しにして次へ進む |
| 今日はスキップ | 今日はこのタスクをやらない（非表示になる） |

**「後で」と「今日はスキップ」の違い：**

| アクション | 動作 | 進捗への影響 |
|------------|------|--------------|
| 完了 | 記録して次へ | 3/5 → 4/5 |
| 後で | 後回しにして次へ（後でまた表示される） | 3/5 のまま |
| 今日はスキップ | 今日は非表示（明日リセット） | 3/5 → 3/4 |

### 管理タブ
習慣の追加・編集・削除・並び替えを行います。

| 要素 | 説明 |
|------|------|
| 絵文字 | タップで絵文字ピッカーを表示 |
| 習慣名 | 直接入力して編集 |
| 🤖ボタン | AIが習慣名に合った絵文字を提案 |
| ×ボタン | 習慣を削除（確認あり） |
| 上下ボタン | 表示順を変更 |
| + 追加 | 新しい習慣を追加 |

### 履歴タブ
過去の記録を一覧で確認できます。

| 要素 | 説明 |
|------|------|
| 並び替えボタン | 新しい順/古い順を切り替え |
| 履歴リスト | 直近100件の記録を表示 |

### 分析タブ
習慣の継続状況をグラフで確認し、AIによる分析を受けられます。

| 要素 | 説明 |
|------|------|
| 期間ボタン | 1W（1週間）/ 1M（1ヶ月）/ 6M（6ヶ月）/ 1Y（1年） |
| 習慣選択 | グラフに表示する習慣を選択 |
| グラフ | 選択した習慣の実行時刻を折れ線で表示 |
| AI分析を実行 | AIが習慣データを分析してアドバイス |

## よくある質問（FAQ）

### Q: データはどこに保存されますか？
A: データはクラウド（Cloudflare Workers KV）に保存されます。同じパスワードでログインすれば、別のデバイスからも同じデータにアクセスできます。

### Q: 記録した時刻を間違えました。修正できますか？
A: はい。記録タブで完了時刻をタップすると、時刻を修正できます。

### Q: 習慣を削除すると過去の記録も消えますか？
A: 習慣を削除しても、過去の記録（ログ）は履歴タブに残ります。ただし、習慣名の表示が「Unknown」になる場合があります。

### Q: AI分析はどのようなデータを使いますか？
A: 定義されている習慣のリストと、直近50件のログデータをAIに送信して分析します。

### Q: オフラインでも使えますか？
A: いいえ。インターネット接続が必要です。

## トラブルシューティング

### ログインできない
- パスワードが正しいか確認してください
- インターネット接続を確認してください
- ブラウザのキャッシュをクリアしてみてください

### データが表示されない
- ページを再読み込みしてみてください
- 別のブラウザで試してみてください

### AI分析が動かない
- しばらく待ってから再度試してください
- 習慣とログが登録されているか確認してください
